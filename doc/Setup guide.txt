1- Download/Install Eclipse IDE for Java EE Developers (https://www.eclipse.org/downloads/)
	-> Requires Java SE Development Kit 8 Downloads (http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
2- a. Right click In project explorer
	-> Import 
	-> Existing Maven projects
   b. Set Root directory
   c. Advanced --> profiles: eclipse
                   Name template: Actimania
3- To run a unit test, open a unit test file (Under actimania\src\server\src\test\java\com\backend\* )
	-> Select the green play button dropdown 
	-> Run As
	-> JUnit Test
4- To start the server (Not yet working)
	a. Download tomcat7
	b. Open a command prompt and go to actimania\projects
	   -> execute command "build.bat server java release" (This starts mongodb)
	c. In eclipse Select the green button dropdown --> Run as --> Maven build
	   -> Goals : tomcat7:run
	d. open http://localhost:8000/backend
